# Server-Utility

このプラグインは、 Bukkit/Spigot 用に開発されたものです。
サーバー運営に役立つ機能を追加し、主にスタッフの方々に向けて開発しました。

使っていただけると喜びます！

かなりの部分を非同期化しているので、サーバーのネットワークに影響する可能性は非常に少ないです。

要件等だった GUI-API は改良しました。
